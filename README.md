## To run this project you should open 2 terminals:

### Terminal for backend:
  1. cd Backend<br>
  2. npm i<br>
  3. node index.js<br>

### Terminal for frontend:
  1. cd frontend\app<br>
  2. npm i<br>
  3. npm start<br>
